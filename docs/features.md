# Features

MiniCiv includes several key features for managing users, licenses, and games.

## Authentication

- Email/password registration and login
- Session management with CSRF protection
- Password reset functionality
- Social login (configurable)

## Dashboard

- User profile management
- License inventory
- Purchase history
- Favorite games
- Admin tools (for admins)

## License Management

- License creation and assignment
- Domain restrictions
- Expiration tracking
- Public validation API
- Purchase system with PayPal/Stripe

## Games

- Game listings by category
- Favorites system
- Curated game list fallback
- Theme integration

## MiniCiv

- Browser-based civilization game
- Save/load game states
- User-specific game progress
- Simple client-side demo

## Shop

- Product catalog
- Purchase flow
- Payment processing

## Admin Panel

- User management
- License management
- Product management
- Server management
- Event logs
- External log ingestion

## API

- License validation endpoints
- JSON responses
- Event log receiver

## Themes

- Multiple frontpage themes (default, games, license, login)
- Responsive design
- Tailwind CSS styling

## Other

- Email testing
- API lab for testing endpoints
- Whois lookup
- Subdomain management
- Certificate management
